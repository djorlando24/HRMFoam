# Ankit water spray nozzle - test case

'Version 2' grid for Ankit's d=0.4mm nozzle with a hemispherical (slightly stretched) outlet plenum.
The grid has approx. 1.2 million cells, generated by snappyHexMesh. Fine refinement inside the nozzle hole.

Set h = 0.110000E+07 and p = 60 bar inlet stagn pressure to obtain 526K (253C) compressed liquid at 0.796673E+03 kg/m3
Set back pressure 1 bar

Estimate viscosity and surface tension at about 250C value, quite low.

DPS updated some of the constants, and a bug fix has been applied to modelCalc.C
